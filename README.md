# Synthetic test fixtures

Every file in this directory is fictional and intended only for automated or manual testing. It must contain no client information, privileged communications, personal information, or confidential material.

Each fixture must:

- be listed in `manifest.json`;
- be visibly labelled `SYNTHETIC` inside the file;
- use fictional people, organizations, addresses, and facts; and
- be reviewed before use in a public demo or recorded test.

Binary-format fixtures can be added when ingestion tests are implemented. Their source text and generation instructions should remain here so the fixture can be audited and reproduced.
